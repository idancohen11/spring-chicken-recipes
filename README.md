# Spring Chicken Recipes

A beautifully designed, mobile-friendly recipe website featuring eight curated spring chicken recipes.

## Design Philosophy

This website embodies an **editorial/magazine aesthetic** that combines:
- Warm, earthy color palette inspired by spring harvest
- Sophisticated serif typography (Cormorant Garamond) paired with clean sans-serif (Work Sans)
- Asymmetric grid layouts with generous whitespace
- Smooth animations and micro-interactions
- Mobile-first responsive design

## Features

- 8 detailed spring chicken recipes with ingredients and instructions
- Expandable recipe cards for clean browsing
- Mobile-friendly touch interactions
- Smooth scroll animations
- Accessibility considerations (reduced motion support)
- Pure HTML/CSS/JS - no build process required

## Recipes Included

1. Herb-Roasted Spring Chicken with Lemon & Thyme
2. Honey-Glazed Spring Chicken with Mustard
3. Mediterranean Spring Chicken with Olives & Tomatoes
4. Asian-Inspired Spring Chicken with Ginger & Soy
5. Garlic Butter Spring Chicken with Herbs
6. Moroccan-Spiced Spring Chicken with Apricots
7. Spring Chicken in White Wine & Cream Sauce
8. Citrus-Glazed Spring Chicken with Orange & Lime
